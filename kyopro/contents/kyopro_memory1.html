A:まあ直角だしねぇ(´^ω^`)
B:コラッツの問題の応用っすかねぇ...範囲保証があるのでシミュレートしていけば解けます。setつかうと楽だと思うぜ。
C:逆に目標の高さになっている花を逆再生して縮めてゆくように考えると楽です。
花の目的の高さの合計と差分を保存していき、+と-の間で処理をしていきます。...説明が下手だね。



4 23 75 0 23 96 50 100
について差分をとっていくと、
4 19 52 -75 23 73 -46 50 -100(最後の値を打ち消すのがポイント！これを忘れてて結構時間食った...)
はじめに保存した目的の高さの合計から(幅*縮めた数)をひいて、合計、つまり残っている花の高さを0にしたらおしまいです。では縮めていきましょう。
ここで正負が変わっている場所を見るのです。例えば、見ていくと52、-75となっているところは、もとの配列を見ると山となります。
そこをくずして(正と負を同じだけ0に近づける...absのminだけ縮めますって言えばいいのかなあ)、
4 19 0 -23 23 73 -46 50 -100となります。(合計から52*2引く)
つぎは
4 0 0 -4 23 73 -46 50 -100 (19*3)
とか
0 0 0 0 23 73 -46 50 -100 (4*4)
とか
0 0 0 0 23 27 0 50 -100 (46*2)
と打ち消していきます。
